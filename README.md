# Receivables-Fund-Simulator
Simulation framework for receivables-backed private credit funds.
A structured finance simulation engine that models receivables portfolios, defaults, recoveries, funding costs, and investor returns.
The platform enables scenario analysis for receivables financing structures and evaluates fund performance under different credit and liquidity conditions.
Built for private credit professionals, SCF platforms, structured finance teams, and alternative asset managers.
Key Features
• Receivables Pool Modeling
• Default & Recovery Simulation
• Cash Flow Waterfall
• IRR Calculation
• Investor Return Analysis
• Scenario & Stress Testing
